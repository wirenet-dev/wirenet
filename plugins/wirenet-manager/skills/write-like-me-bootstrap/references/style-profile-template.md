# Style Profile Shape

Create `~/.agents/skills/write-like-me/references/style-profile.md` with only
evidence-grounded rules and synthetic examples.

## Source Basis

- Approved source scopes
- Date range
- Confidence
- Important gaps

## Global Voice

- Default stance
- Pacing
- Directness
- Warmth
- Humor
- Formality
- Common moves
- Avoid

## Postures

For each supported posture, record when to use it, shape, patterns, things to
avoid, and one synthetic example:

- quick chat reply;
- decision or pushback;
- delegation or ask;
- executive or status update;
- email reply;
- intro or relationship note;
- scheduling or admin;
- apology, correction, or repair.

## Critique And Update Rules

Capture the most common ways a draft becomes too formal, verbose, generic, soft,
sharp, or context-poor. Update posture rules after user corrections; never add
raw private excerpts.
