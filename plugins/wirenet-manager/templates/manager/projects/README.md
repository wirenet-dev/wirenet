# Projects

This folder holds open Project Packs. Each packet begins with `README.md` for
canonical state and `AGENTS.md` for local routing. Agents may add optional goals,
results, worklogs, OKF logs, indexes, decisions, or other concepts when those
documents improve the handoff.

Use [the OKF project index](index.md) for progressive disclosure. This README is
retained as the human and Jason-compatible collection guide; the index is an
additive machine- and human-readable projection.
