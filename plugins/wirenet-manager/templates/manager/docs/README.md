# Docs

An optional shelf for structured, durable documents that do not belong more
clearly in a Project Pack, person note, or curated source. Agents may create and
organize documents here when the distinction from quick `notes/` is useful; the
Manager imposes no fixed taxonomy.

Portable documents use an OKF `type` and ordinary Markdown links. Product or
project documentation should remain with its external repository when that is
the stronger canonical home.
